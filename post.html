Hari Ini Aku Belajar Animasi Scroll Lagi

19 Februari 2026 4 menit baca

Ini adalah isi artikel contoh yang sudah rapi. Kamu bisa ganti teks ini dengan konten asli kamu. Style glassmorphism-nya tetap konsisten dengan halaman utama.

Kenapa pakai Intersection Observer?

Karena jauh lebih efisien daripada mendengarkan event scroll setiap pixel. Browser nggak perlu kerja keras terus-menerus.

// Contoh sederhana
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.classList.add('visible');
    }
  });
}, { threshold: 0.1 });

document.querySelectorAll('.fade-in').forEach(el => observer.observe(el));

Sisanya tinggal tambah sesuai kebutuhan. Semoga tampilannya sekarang lebih clean dan nggak acak-acakan lagi ya!
